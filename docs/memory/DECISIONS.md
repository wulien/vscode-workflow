# 决策记录 (ADR)

> 记录关键技术决策及其原因，避免重复讨论已做的决策。

## 决策模板

```markdown
### ADR-XXX: [决策标题]

**日期**: YYYY-MM-DD
**状态**: [提议 | 已采纳 | 已废弃 | 已替代]

**背景**: 
[为什么需要做这个决策]

**选项**:
1. [选项A] — [优缺点]
2. [选项B] — [优缺点]
3. [选项C] — [优缺点]

**决定**:
[选择了什么，为什么]

**影响**:
[这个决策带来的影响]
```
